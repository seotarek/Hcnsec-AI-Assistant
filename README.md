# Hcnsec AI Assistant 🤖

A lightweight, powerful AI coding assistant for VS Code / Antigravity powered by the Hcnsec API.

## Features
- **Sidebar Chat:** A dedicated sidebar panel to chat with AI directly in your editor.
- **Context-Aware Coding:** Select any code, right-click, and choose "Hcnsec: Ask AI" to instantly send it to the chat for explanations, refactoring, or bug fixing.
- **Multiple Models Supported:** Easily switch between top-tier models like DeepSeek, Qwen, GLM, Kimi, and more, straight from the sidebar settings!
- **Bring Your Own Key (BYOK):** Use your own Hcnsec API key securely.

## Installation
1. Download the `.vsix` release file.
2. Open VS Code / Antigravity.
3. Go to the Extensions view.
4. Click on the `...` menu and select **Install from VSIX...**
5. Select the downloaded file.

## Configuration
1. Open the Hcnsec AI sidebar (look for the robot icon).
2. Enter your API Key.
3. Select your preferred Model.
4. Click **Save Settings**.

## Usage
- Start chatting directly in the sidebar.
- Select code in any editor, right-click, and select **Hcnsec: Ask AI (Selected Text)**.
